# Adding to your project

1. Add the JitPack repository to your build.gradle at the end of repositories:

```groovy
repositories {
  ...
  maven { url 'https://jitpack.io' }
}
```

2. Add the dependency

```groovy
implementation 'com.github.ctrlbytes:CodeKit:0.1.1'
```
